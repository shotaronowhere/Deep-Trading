# Archive

This directory stores superseded or historical documents.

Use archive docs for context and audit trail, not as implementation truth.

Current archive layout:

- `docs/archive/rebalancer/`: benchmark reports and legacy rebalancer investigations
- `docs/archive/implementation/`: implementation migration/change logs
- `docs/archive/research/`: exploratory notes and long-form improvement brainstorming
- `docs/archive/changelog/`: one-off historical change notes
- `docs/archive/background/`: superseded explanatory docs merged into canonical specs
- `docs/archive/components/`: superseded component-level docs merged into canonical docs
- `docs/archive/plans/`: completed implementation plans

Canonical behavior and policy remain in:

- `docs/waterfall.md`
- `docs/rebalancer.md`
- `docs/rebalancer_mixed.md`
- `docs/rebalancer_approaches_playbook.md`
- `docs/rebalancer_policy_metrics_schema.md`
- `docs/model.md`
- `docs/architecture.md`
